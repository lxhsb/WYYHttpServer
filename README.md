# WYYHttpServer

一个简单的HttpServer  
写这个是为了学习网络编程，目前正在阅读《HTTP权威指南》，看一点实现一点。

主要实现了
+ 单站点静态文件(UTF8)访问支持
+ 阻塞/非阻塞模式
+ 长连接(非阻塞模式)
+ 简单日志功能
